# e-Penomoran Kantor Kementerian Agama Kota Depok

Sistem Informasi Manajemen Penomoran dan Pengarsipan Surat secara Digital dari Urusan Umum Sub Bagian Tata Usaha Kantor Kementerian Agama Kota Depok. Aplikasi ini dirancang untuk memfasilitasi manajemen administrasi persuratan secara terpusat, transparan, dan efisien. 

**Akses Aplikasi:** [https://depok.kemenag.go.id/s/penomoran](https://depok.kemenag.go.id/s/penomoran)

---

## Fitur Utama

* **Manajemen Nomor Surat:** Penomoran otomatis untuk berbagai kategori surat dinas.

## Teknologi yang Digunakan

* **Bahasa Pemrograman:** JavaScript
* **Database:** Spreadsheet
* **Frontend:** HTML5, CSS3, JavaScript
* **URL:** [https://depok.kemenag.go.id/s/penomoran](https://depok.kemenag.go.id/s/penomoran)

## Instalasi Lokal

1.  **Clone Repositori**
    ```bash
    Silakan ditanyakan ke Urusan Umum Sub Bagian Tata Usaha Kantor Kementerian Agama Kota Depok.
    ```

2.  **Konfigurasi Database**
    * Buat database baru di Google Spreadsheet.
    * Import file `Spreadsheet` yang tersedia (jika ada).
    * Sesuaikan kredensial database pada file `Kode.gs` atau file konfigurasi terkait.

3.  **Jalankan Aplikasi**
    * Akses melalui browser: `pranala yang diberikan oleh Google Apps Script`.

## Informasi Tambahan

* **Pembuatan Akun & Reset Password:** Untuk unit kerja di lingkungan Kantor Kementerian Agama Kota Depok, silakan menghubungi **Urusan Umum**.
* **Maintenance:** Pembaruan sistem dilakukan secara berkala untuk meningkatkan keamanan dan performa layanan.
